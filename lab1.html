What is HTML?

HTML stand for Hypertext Markup Language, and it is responsible for displaying webpages. The most
recent version of HTML currently is HTML5.

What is a Web Server?

A Web Server is a computer connected to internet that has the main purpose of waiting for requests
from web browsers. Some of these requests include displaying web pages, images, movies, and videos.

What is ARPANET?

ARPANET, which was created in 1969, was essentially the first version of the internet, which is an
interconnection of computers communicating using a set of rules. The network was created by ARPA
(Advanced Research Projects Agency), who were funded by the USA Department of Defense. Originally,
ARPANET had four hosts at universities in Western US, and 128 host computers.

What is WWW?

WWW, or the World Wide Web, is a software application that makes use of the internet's vast
infrastructure and allowed computers to access documents of different kinds all over the interent.
It was created around 1990 by Tim Berners-Lee while he worked at CERN Nuclear Physics Research Lab
in Geneva Switzerland.

What is Domain Name?

Domain name is essentially the name of a website. It is a virtual property that that can be bought
and owned by people.

What is http?

http stands for "hypertext transfer protocol," and is a protocol that specifies how web pages are
delivered. The secure version of http is https, and all protocols are a primary component in URLs.

What is W3C?

W3C, which stands for World Wide Web Consortium. The non-profit organization helped to end the browser
wars between the Microsoft and Netscape by releasing uniform standards for HTML and other web-related
technologies. It allowed browsers to work mostly the same way, and helped to mitigate immense competition.
W3C was created by Tim-Berners and others who helped create WWW. Major corporations like Microsoft and
Netscape also got seats at the table because their input was important.

What is URL?

URL, which stands for Uniform Resource Locator, has three primary components: the how, where, and what.
The "how" is protocols like http and https that specify how web pages are delivered, as well as others
such as ftp, which initiate file downloads instead of temporary web page loading. The "where" is usually a
named address, but can also be a raw numeric IP address, which is the identity number every machine, software,
or website has. The "what" specifies a specific resource on the server, and also descends into subfolders
within the web site.
